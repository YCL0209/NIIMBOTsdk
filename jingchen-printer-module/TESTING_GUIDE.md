- 打印机输出一张条码标签
- 条码内容清晰可读

**打印流程验证（官方 5 步流程）：**
```
1. 正在打印条码: TEST-001
2. [内部] startJob 成功        ← 步骤1: 开始任务，设置打印参数
3. [内部] InitDrawingBoard 成功 ← 步骤2: 初始化画板尺寸
4. [内部] DrawLableBarCode 成功 ← 步骤3: 绘制条码
5. [内部] commitJob 成功        ← 步骤4: 提交打印
6. [内部] endJob 成功           ← 步骤5: 结束任务
7. 条码打印成功!
```

⚠️ **重要说明**：
- 这是官方推荐的打印流程，所有打印操作必须遵循此流程
- `print()` 方法已废弃，不应在新代码中使用
- LabelType 枚举值已更新（1-6, 10），请使用新的枚举值

**可能的错误：**
| 错误 | 原因 | 解决方法 |
|------|------|---------|
| `条码内容不能为空` | 未输入内容 | 输入有效内容 |
| `操作失败` | 打印机状态异常 | 重新连接打印机 |
| 打印出空白纸 | 标签尺寸不匹配 | 检查纸张和参数 |

### 测试用例 15：取消打印测试 ✨新增

**目标：** 验证 `cancelJob()` API 取消正在进行的打印任务

**前置条件：** 打印机已连接

**测试步骤：**

#### 场景 A：启动任务后立即取消
```typescript
try {
  // 1. 开始一个打印任务
  await printer.startJob(3, LabelType.GAP_PAPER, 1, 1);
  await printer.initBoard({ width: 40, height: 20, rotate: RotateAngle.ROTATE_0 });
  await printer.drawBarcode({
    x: 5, y: 5, width: 30, height: 10,
    value: 'CANCEL-TEST',
    codeType: BarcodeType.CODE128,
    rotate: RotateAngle.ROTATE_0,
    fontSize: 3, textHeight: 3, textPosition: TextPosition.BOTTOM
  });

  // 2. 提交打印
  await printer.commitJob(1);

  // 3. 立即取消
  await printer.cancelJob();
  console.log('打印任务已取消');

} catch (error) {
  console.error('取消失败:', error);
}
```

#### 场景 B：批量打印中途取消
```typescript
// 开始打印 10 页
await printer.startJob(3, LabelType.GAP_PAPER, 1, 10);

for (let i = 0; i < 10; i++) {
  await printer.initBoard({ width: 40, height: 20, rotate: RotateAngle.ROTATE_0 });
  await printer.drawText({
    x: 10, y: 10, width: 30, height: 10,
    value: `Page ${i + 1}`,
    rotate: RotateAngle.ROTATE_0,
    fontSize: 5
  });
  await printer.commitJob(1);

  // 打印 3 页后取消
  if (i === 2) {
    await printer.cancelJob();
    console.log('批量打印已取消');
    break;
  }
}
```

**期望结果：**
- 日志显示：`[内部] cancelJob 成功`
- 打印机停止打印
- 当前任务被取消
- 可以开始新的打印任务

**可能的场景：**
| 场景 | 结果 | 说明 |
|------|------|------|
| commitJob 后立即取消 | 可能已打印 | 取决于打印机响应速度 |
| 多页打印中途取消 | 部分页已打印 | 后续页不再打印 |
| 未开始任务就取消 | 返回错误 | 无任务可取消 |

```typescript
  fontSize: 3, textHeight: 3, textPosition: TextPosition.BOTTOM
});

await printer.drawQRCode({
  x: 10, y: 16, width: 12, height: 12,
  value: 'https://example.com',
  codeType: QRCodeType.QR_CODE,
  rotate: RotateAngle.ROTATE_0
});

// 3. 生成预览（displayScale: 显示比例，建议 1-3）
const base64Image = await printer.generateImagePreviewImage(2);

// 4. 显示预览
document.getElementById('preview').innerHTML =
  `<img src="data:image/png;base64,${base64Image}" />`;

// 5. 继续打印或取消
await printer.commitJob(1);
await printer.endJob();
```

## 🐛 常见问题排查

### 问题 1：服务连接失败

**症状：** 点击连接按钮后显示"连接超时"

**检查清单：**
- [ ] 打印服务是否已启动？运行安装目录中的服务程序
- [ ] 端口 37989 是否被占用？`netstat -ano | findstr :37989`
- [ ] 防火墙是否阻止连接？尝试关闭防火墙测试
- [ ] WebSocket 地址是否正确？默认 `ws://127.0.0.1:37989`

### 问题 2：扫描不到打印机

**症状：** 显示"找到 0 台USB打印机"

**检查清单：**
- [ ] USB 线是否正常连接？
- [ ] 设备管理器中是否显示打印机？
- [ ] 打印机电源是否开启？
- [ ] 是否安装了打印机驱动？查看官方驱动下载页面
- [ ] 尝试重新插拔 USB 线

### 问题 3：打印出空白纸

**症状：** 打印机走纸但没有内容

**检查清单：**
- [ ] 标签尺寸设置是否与实际纸张匹配？
- [ ] 打印浓度是否太低？默认值为 3（1-15）
- [ ] 条码内容是否有效？检查输入内容
- [ ] 纸张是否正确安装？查看打印机说明书

### 问题 4：条码无法扫描识别

**症状：** 打印的条码扫描枪读不出来

**检查清单：**
- [ ] 条码内容是否符合类型要求？（如 EAN13 必须 13 位数字）
- [ ] 条码尺寸是否太小？建议最小宽度 30mm
- [ ] 打印浓度是否合适？尝试调整浓度
- [ ] 标签材质是否合适？使用热敏纸
- [ ] 扫描枪是否支持该条码类型？

### 问题 5：批量打印中断

**症状：** 批量打印只打印了部分就停止

**检查清单：**
- [ ] 纸张是否充足？
- [ ] 打印机是否过热保护？等待冷却后继续
- [ ] 查看日志中的错误信息
- [ ] 检查 USB 连接是否稳定
